## all even

Write a function, _allEven_, that takes in an array of numbers as an argument. The function should return a boolean
indicating whether or not every element of the array is an even number.

You can assume that the array is non-empty.

#### test_00:

```js
allEven([4, 90, 68, 6, -2]); // -> true
```

#### test_01:

```js
allEven([14, 40, 36, 3]); // -> false
```

#### test_02:

```js
allEven([30, 24, 2048, 0, 12, 50]); // -> true
```

#### test_03:

```js
allEven([7, 7, 7, 7]); // -> false
```

#### test_04:

```js
allEven([100]); // -> true
```

#### test_05:

```js
allEven([1, 2, 4, 6, 8]); // -> false
```

#### test_06:

```js
allEven([42, 18, 96, 4, 70, 12, 58, 30, 84, 26]); // -> true
```
