## longest unique substring

Write a function that takes in a string as an argument. The function should return the length of the longest substring
that consists of only unique characters.

#### test_00:

```js
longestUniqueSubstring("abcabcqbb"); // -> 4
// 'abcq' is the longest substring with unique characters and its length is 4
```

#### test_01:

```js
longestUniqueSubstring("forgeonwards"); // -> 9
```

#### test_02:

```js
longestUniqueSubstring("tmmzuxt"); // -> 5
```

#### test_03:

```js
longestUniqueSubstring("racecarisgreatness"); // -> 8
```

#### test_04:

```js
longestUniqueSubstring("zzzzzz"); // -> 1
```

#### test_05:

```js
longestUniqueSubstring("zqzzzz"); // -> 2
```
